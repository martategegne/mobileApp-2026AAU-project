# Smart Study Resource Sharing Platform

## Description
A mobile application that allows students to share and access academic resources such as lecture notes, past exams, and study materials.

## Features
- Authentication (Signup/Login)
- Authorization (Student/Admin roles)
- Resource Management (CRUD)
- Resource Requests (CRUD)
- Offline support
- Search functionality

## Architecture
- Domain Driven Design (DDD)
- REST API based backend
- Flutter frontend

## Technologies
- Flutter (Frontend)
- REST API (Backend)
- Local Database

## Testing
- Unit Testing
- Widget Testing
- Integration Testing
